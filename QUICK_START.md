# 에코트래커 빠른 시작 가이드

## 🚀 5분 안에 시작하기

### 1단계: 프론트엔드 실행

```bash
cd client
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`을 열면 에코트래커를 사용할 수 있습니다!

### 2단계: 백엔드 실행 (선택사항)

현재 MVP는 프론트엔드만으로도 작동합니다 (임시 데이터 사용). 
API를 연결하려면:

```bash
cd server
npm install
npm run dev
```

백엔드는 `http://localhost:3000`에서 실행됩니다.

## 📱 사용 방법

### 첫 방문
1. 온보딩 슬라이드를 확인하세요
2. 닉네임을 설정하거나 건너뛰기
3. 홈 화면에서 "체크 시작하기" 클릭

### 일회용품 체크
1. 카테고리 탭에서 음식/음료/포장재 선택
2. 오늘 사용한 항목을 선택하고 수량 조절
3. 기타 항목은 수동으로 입력
4. "체크 완료하기" 버튼 클릭

### 결과 확인
- 환경 영향도 (나무, CO2, 해양 영향)
- 일상 비교 (스마트폰 충전, 자동차 주행)
- 제로 데이 달성 시 축하 메시지!

### 통계 보기
- 주간/월간/전체 기간 통계
- 요일별 사용 패턴
- 카테고리별 분석
- 누적 환경 기여도

### 랭킹 확인
- **에코 챔피언**: 사용량이 적을수록 높은 순위
- **챌린저 스타**: 전주 대비 개선율이 높을수록 높은 순위
- 내 순위 및 상위 퍼센트 확인

## 🔧 문제 해결

### 프론트엔드가 시작되지 않을 때
```bash
cd client
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### 백엔드가 시작되지 않을 때
```bash
cd server
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### 포트가 이미 사용 중일 때
- 프론트엔드: `vite.config.ts`에서 포트 변경
- 백엔드: `.env` 파일에서 PORT 변경
