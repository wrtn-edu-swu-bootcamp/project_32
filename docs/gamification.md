# 게임화 전략 (Gamification Strategy)

## 개요
에코트래커의 게임화 전략은 사용자가 환경 보호 활동을 지속적이고 즐겁게 실천할 수 있도록 설계되었습니다. 포인트, 레벨, 뱃지, 칭호 등 다층적인 보상 시스템을 통해 장기적인 참여를 유도합니다.

---

## 1. 전체 게임화 구조

```
사용자 행동 (체크, 감소 등)
         ↓
즉각적 보상 (포인트 획득)
         ↓
누적 시스템 (레벨업, 뱃지 획득)
         ↓
사회적 인정 (랭킹, 칭호)
         ↓
지속적 참여 (새로운 목표 제시)
```

## 2. 포인트 시스템

### 2.1 포인트 획득 방법

#### 기본 활동 포인트

**매일 체크**
- 체크 완료: +10p
- 상황: 하루 기록을 완료할 때마다
- 메시지: "오늘도 기록 완료! +10p"

**연속 체크 보너스**
```
2일 연속: +5p
3일 연속: +5p
7일 연속: +15p (누적 +25p)
14일 연속: +20p (누적 +45p)
30일 연속: +30p (누적 +75p)
100일 연속: +100p
```
- 메시지: "🔥 7일 연속 체크! 열정이 대단해요 +15p"

#### 성과 기반 포인트

**일회용품 감소**
- 전날 대비 감소: +20p
- 전주 대비 20% 이상 감소: +50p
- 전월 대비 30% 이상 감소: +100p
- 메시지: "어제보다 2개 줄였어요! 멋져요 +20p"

**제로 데이 달성**
- 하루 일회용품 0개: +50p
- 주 2회 제로 데이: +100p (보너스)
- 주 3회 이상 제로 데이: +200p (보너스)
- 메시지: "🌟 제로 데이 달성! 완벽해요 +50p"

**카테고리별 제로**
- 음식 관련 제로: +15p
- 음료 관련 제로: +15p
- 포장재 제로: +15p
- 메시지: "오늘 음료 관련 일회용품 0개! +15p"

#### 뱃지 획득 포인트

- 새로운 뱃지 획득: +100p
- 희귀 뱃지 획득: +300p
- 전설 뱃지 획득: +500p
- 메시지: "🏅 새 뱃지 획득! +100p"

#### 랭킹 진입 포인트

- 에코 챔피언 Top 100: +50p (주간)
- 에코 챔피언 Top 10: +200p (주간)
- 챌린저 스타 Top 100: +50p (주간)
- 챌린저 스타 Top 10: +200p (주간)
- 메시지: "🏆 Top 100 진입! +50p"

#### 마일스톤 포인트

**누적 체크**
```
10회 체크: +50p
50회 체크: +200p
100회 체크: +500p
365회 체크: +1,000p
```

**누적 절감**
```
나무 10그루: +100p
나무 50그루: +300p
나무 100그루: +500p
나무 500그루: +1,500p
```

### 2.2 포인트 활용

포인트는 레벨업의 경험치로만 사용되며, 소비되지 않고 누적됩니다.

**레벨업 시스템**
- 획득한 포인트 = 레벨 경험치
- 레벨이 오를수록 더 많은 포인트 필요
- 레벨은 절대 내려가지 않음 (안정성)

**월간 포인트 랭킹**
- 매월 획득한 포인트로 별도 랭킹
- 월말 상위 랭커에게 특별 칭호
- 다음 달 포인트 누적은 리셋되지 않음

## 3. 레벨 시스템

### 3.1 레벨 단계

```
Lv.1  새싹 환경러          0 - 500p
Lv.2  초보 실천러          500 - 1,500p
Lv.3  그린 챌린저          1,500 - 3,000p
Lv.4  에코 워리어          3,000 - 6,000p
Lv.5  환경 수호자          6,000 - 10,000p
Lv.6  지구 지킴이          10,000 - 20,000p
Lv.7  에코 마스터          20,000 - 35,000p
Lv.8  그린 레전드          35,000 - 55,000p
Lv.9  플래닛 세이버        55,000 - 80,000p
Lv.10 얼스 가디언         80,000p+
```

### 3.2 레벨별 특징 및 혜택

#### Lv.1 새싹 환경러 (0-500p)
**소요 기간**: 약 2-3주
**특징**: 환경 보호의 첫 발을 내딛은 사용자
**혜택**:
- 기본 기능 이용
- 기본 통계 제공
- 일일 체크 기능

**레벨업 메시지**:
```
🌱 새싹이 돋아났어요!
레벨 2: 초보 실천러

작은 씨앗이 자라나고 있어요
계속 물을 주세요!
```

#### Lv.2 초보 실천러 (500-1,500p)
**소요 기간**: 약 1-2개월
**특징**: 체크 습관이 잡힌 사용자
**혜택**:
- 주간 상세 리포트
- 카테고리별 분석
- 커스텀 프로필 아이콘 3종
- 개인 목표 설정 기능

#### Lv.3 그린 챌린저 (1,500-3,000p)
**소요 기간**: 약 2-3개월
**특징**: 적극적으로 일회용품을 줄이는 사용자
**혜택**:
- 월간 상세 리포트
- 전체 기간 통계
- 커스텀 프로필 아이콘 5종
- 랭킹 하이라이트 표시
- 비교 분석 기능

#### Lv.4 에코 워리어 (3,000-6,000p)
**소요 기간**: 약 3-5개월
**특징**: 환경 보호를 위해 싸우는 전사
**혜택**:
- 예측 분석 기능
- AI 맞춤 조언
- 커스텀 프로필 아이콘 8종
- 특별 칭호 부여
- 프리미엄 통계 대시보드

#### Lv.5 환경 수호자 (6,000-10,000p)
**소요 기간**: 약 6-9개월
**특징**: 환경을 지키는 수호자
**혜택**:
- 연간 리포트
- 고급 인사이트
- 커스텀 프로필 전체 (15종)
- 월간 MVP 후보 자격
- 커뮤니티 리더 자격

#### Lv.6 지구 지킴이 (10,000-20,000p)
**소요 기간**: 약 10개월-1년
**특징**: 지구를 지키는 베테랑
**혜택**:
- 베타 기능 우선 체험
- 특별 이벤트 우선 참여
- 전용 뱃지 & 칭호
- 인플루언서 자격

#### Lv.7 에코 마스터 (20,000-35,000p)
**소요 기간**: 약 1.5년
**특징**: 환경 보호의 달인
**혜택**:
- 마스터 전용 뱃지
- 에코 마스터 칭호
- 전용 프로필 테마
- 공식 인증서 발급

#### Lv.8 그린 레전드 (35,000-55,000p)
**소요 기간**: 약 2년
**특징**: 전설이 된 환경 지킴이
**혜택**:
- 레전드 전용 컨텐츠
- 특별 프로필 이펙트
- 커뮤니티 명예의 전당
- 기관 협력 기회

#### Lv.9 플래닛 세이버 (55,000-80,000p)
**소요 기간**: 약 2.5-3년
**특징**: 행성을 구하는 영웅
**혜택**:
- 최상위 칭호
- 전용 애니메이션 효과
- 언론 인터뷰 기회
- 환경 캠페인 참여

#### Lv.10 얼스 가디언 (80,000p+)
**소요 기간**: 3년+
**특징**: 지구의 수호신, 최고 레벨
**혜택**:
- 영구 전설 칭호
- 특별 기념품
- 공식 홍보대사 자격
- 환경부 장관 표창 추천
