# 일회용품 카테고리 및 데이터

## 개요
이 문서는 에코트래커에서 다루는 일회용품의 상세 목록과 각 항목의 환경 영향도 데이터를 정의합니다. 사용자가 체크하는 선택지와 환경 영향 계산의 기반이 되는 핵심 데이터입니다.

---

## 1. 일회용품 카테고리 구조

에코트래커는 3가지 주요 카테고리를 중심으로 운영됩니다:

1. **음식 관련** - 배달, 테이크아웃, 외식 시 사용되는 일회용품
2. **음료 관련** - 카페, 편의점 등에서 구매하는 음료 용기
3. **포장재** - 쇼핑, 택배 등에서 발생하는 포장 관련 일회용품

## 2. 음식 관련 일회용품

## 5. 환경 영향도 점수 시스템

### 5.1 점수 산정 기준

**1점**: 영향 최소
- 분해 기간 짧음 (1년 이하)
- CO2 배출량 낮음 (10g 이하)
- 재활용 가능
- 해양 영향 없음

**2점**: 영향 약간
- 분해 기간 짧음 (5년 이하)
- CO2 배출량 약간 (10-30g)
- 재활용 가능하나 복잡

**3점**: 영향 보통
- 분해 기간 보통 (5-50년)
- CO2 배출량 보통 (30-50g)
- 재활용 어려움

**4점**: 영향 높음
- 분해 기간 김 (50-500년)
- CO2 배출량 높음 (50-100g)
- 재활용 불가
- 해양 오염 가능성

**5점**: 영향 매우 높음
- 분해 기간 매우 김 (500년+)
- CO2 배출량 매우 높음 (100g+)
- 재활용 불가
- 해양 생태계 위협

### 5.2 점수별 비중

사용자의 일일 환경 영향도는 각 항목의 점수 합계로 계산:

```
일일 환경 영향도 = Σ(항목별 점수 × 사용 개수)

예시:
- 플라스틱 컵 2개 (4점 × 2) = 8점
- 비닐봉지 1개 (4점 × 1) = 4점
- 종이컵 1개 (3점 × 1) = 3점
─────────────────────────────
총 15점
```

## 6. CO2 배출량 환산

### 6.1 일상 비교 단위

사용자에게 친숙한 단위로 변환하여 제공:

**스마트폰 충전**
- 1회 충전 = 약 2g CO2
- 예: 90g CO2 = 스마트폰 45번 충전

**승용차 주행**
- 1km 주행 = 약 150g CO2
- 예: 450g CO2 = 승용차 3km 주행

**나무의 하루 산소 생산량**
- 소나무 1그루 = 약 150g CO2 흡수/일
- 예: 450g CO2 = 소나무 3그루 1일 생산량

**전등 사용**
- 1시간 = 약 50g CO2
- 예: 450g CO2 = LED 전등 9시간 사용

### 6.2 누적 효과 시각화

**주간 누적**
```
이번 주 CO2 절감: 2.1kg
= 승용차 14km 주행량
= 소나무 14그루 1일분
```

**월간 누적**
```
이번 달 CO2 절감: 9.5kg
= 승용차 63km 주행량
= 소나무 63그루 1일분
= 성인 3명의 1일 호흡량
```

**연간 누적**
```
올해 CO2 절감: 115kg
= 승용차 767km 주행량
= 소나무 767그루 1일분
= 비행기 서울-부산 왕복 승객 1명분
```

## 7. 분해 기간 시각화

### 7.1 시간 비교

사용자가 체감할 수 있는 시간 단위로 표현:

**짧은 기간 (1년 미만)**
- "당신의 생일 케이크 초가 사라지기 전에 분해돼요"

**중간 기간 (1-20년)**
- "초등학교 입학부터 대학 졸업까지보다 오래 남아요"
- 나무 젓가락: "태어난 아기가 성인이 될 때까지"

**긴 기간 (20-100년)**
- 종이컵: "여러분의 증손자가 태어날 때까지"

**매우 긴 기간 (100-500년)**
- 플라스틱 컵: "조선 시대에 버렸다면 지금도 있어요"

**초장기 (500년+)**
- 비닐봉지: "1500년대 중세 시대부터 지금까지의 시간"

### 7.2 세대 단위 표현

```
20년 = 1세대
100년 = 5세대
450년 = 22세대
1000년 = 50세대

"이 플라스틱 컵은 당신으로부터
22세대 후손까지 남아있을 거예요"
```
