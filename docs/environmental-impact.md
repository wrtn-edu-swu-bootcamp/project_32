# 환경 영향도 계산 및 시각화

## 개요
에코트래커는 사용자의 일회용품 사용량을 단순한 숫자가 아닌, **구체적이고 체감 가능한 환경 영향**으로 변환하여 보여줍니다. 이를 통해 사용자는 자신의 작은 실천이 지구에 미치는 긍정적인 영향을 실감할 수 있습니다.

---

## 1. 환경 영향도 계산 원칙

### 1.1 기본 철학

```
추상적 → 구체적
"환경에 좋아요" → "나무 3그루를 살렸어요"

복잡함 → 단순함
"LCA 분석 결과..." → "승용차 5km 주행량"

부정적 → 긍정적
"CO2 450g 발생" → "나무 3그루 1일 산소량"

거대함 → 일상적
"지구 온난화" → "스마트폰 45번 충전"
```

### 1.2 변환 단위 체계

모든 일회용품 사용은 다음으로 변환:

1. **나무 (Trees)** - 산소 생산 / CO2 흡수
2. **CO2 무게 (kg)** - 탄소 발자국
3. **일상 비교** - 스마트폰 충전, 승용차 주행
4. **해양 영향** - 플라스틱 오염 지수
5. **자원 절약** - 재활용 가능 자원량

## 2. 나무 환산 시스템

### 2.1 기준치

**소나무 1그루 기준**
- 하루 CO2 흡수량: 약 150g
- 연간 CO2 흡수량: 약 55kg
- 하루 산소 생산량: 약 110g
- 성목 30년생 기준

### 2.2 계산 공식

```
일회용품 CO2 배출량 = Σ(각 항목 CO2)

나무 그루 수 = 총 CO2 배출량 ÷ 150g

예시:
- 플라스틱 컵 2개 (45g × 2) = 90g
- 비닐봉지 1개 = 15g
- 배달 용기 2개 (90g × 2) = 180g
────────────────────────────
총 285g CO2

나무 = 285g ÷ 150g = 1.9그루
→ 사용자에게 "나무 1.9그루"로 표시
```

### 2.3 표현 방식

**단일 데이터 (하루)**
```
오늘 사용한 일회용품
→ 나무 0.3그루 = 1일 산소량
```

**누적 데이터 (주간)**
```
이번 주
→ 나무 2.1그루 보호

이는 어른 1명이 하루 필요한
산소량의 2배예요
```

**누적 데이터 (월간)**
```
이번 달
→ 나무 9그루 보호

작은 정원을 만든 것과 같아요 🌳
```

**누적 데이터 (연간)**
```
올해
→ 나무 115그루 보호

작은 숲을 만들었어요! 🌲🌲🌲
이는 농구장 크기의 숲이에요
```

## 3. CO2 배출량 환산

### 3.1 일상 비교 단위

#### 스마트폰 충전
```
스마트폰 1회 충전 = 약 2g CO2

예시:
90g CO2 = 스마트폰 45번 충전
450g CO2 = 스마트폰 225번 충전
```

**메시지 예시**
- "오늘 사용량은 스마트폰 45번 충전과 같아요"
- "이번 주로 스마트폰 300번 충전량을 절약했어요"

#### 승용차 주행
```
승용차 1km 주행 = 약 150g CO2

예시:
450g CO2 = 승용차 3km 주행
1,500g CO2 = 승용차 10km 주행
```

**메시지 예시**
- "승용차로 집에서 회사까지 거리예요 (5km)"
- "서울-부산 왕복 거리만큼 절약했어요 (900km)"

#### 전등 사용
```
LED 전등 1시간 = 약 50g CO2

예시:
450g CO2 = LED 전등 9시간
```

**메시지 예시**
- "하루 종일 전등을 켜놓은 것과 같아요"
- "한 달간 전등 30시간 사용량 절약"

#### TV 시청
```
TV 1시간 = 약 88g CO2

예시:
450g CO2 = TV 약 5시간 시청
```

**메시지 예시**
- "영화 2편 볼 수 있는 에너지예요"
- "한 주간 TV 10시간 절약"

## 4. 해양 플라스틱 영향 지수

### 4.1 해양 영향 점수 시스템

각 일회용품마다 해양 오염 위험도 점수 부여:

```
점수 산정 기준:
1. 플라스틱 종류 (해양 분해 난이도)
2. 크기 및 무게 (해양 생물 위협도)
3. 미세플라스틱 생성 가능성
4. 해양 유입 가능성
```

**항목별 점수**
```
플라스틱 빨대: 5점 (해양 생물 직접 위협)
플라스틱 컵: 8점 (크고 분해 어려움)
비닐봉지: 10점 (해파리로 오인, 질식 위험)
페트병: 8점 (부피 크고 떠다님)
배달 용기: 7점
일회용 수저: 4점
```

### 4.2 해양 영향 시각화

**일간**
```
오늘의 해양 영향
🌊 해양 플라스틱 위험도 +15점

이는:
🐢 바다거북 1마리가 위험한 수준
🐟 물고기 3마리가 삼킬 수 있는 양
```

**주간**
```
이번 주 해양 보호
🌊 해양 플라스틱 위험도 -42점

당신의 실천으로:
🐋 고래 1마리의 먹이 경로가 안전해졌어요
🌊 해변 2m를 깨끗하게 지켰어요
```

**월간**
```
이번 달 해양 보호
🌊 해양 플라스틱 위험도 -185점

이는:
🏖️ 해변 10m 청소한 것과 같아요
🐠 물고기 50마리를 구했어요
🌊 바다 생태계를 지켰어요
```

### 4.3 메시지 예시

**긍정적 메시지 (절감 시)**
- "오늘 바다가 조금 더 깨끗해졌어요 🌊"
- "바다거북 1마리를 구했어요 🐢"
- "물고기들이 고마워해요 🐟"
- "해변이 더 아름다워졌어요 🏖️"

**정보성 메시지 (사용 시)**
- "오늘 사용한 플라스틱은 바다에서 450년 남아요"
- "비닐봉지 1개는 바다거북에게 치명적이에요"

## 5. 자원 절약 환산

### 5.1 재활용 가능 자원량

**플라스틱 (kg)**
```
계산: Σ(플라스틱 항목 중량)

예시:
플라스틱 컵 2개 (15g × 2) = 30g
페트병 1개 = 25g
비닐봉지 2개 (5g × 2) = 10g
─────────────────────────
총 65g = 0.065kg

주간: 약 0.5kg
월간: 약 2kg
연간: 약 24kg
```

**메시지**
- "이번 달 플라스틱 2kg 절약 = 페트병 80개"
- "올해 플라스틱 24kg 절약 = 대형 쓰레기봉투 8개"

**종이 (kg)**
```
종이컵, 종이 용기, 택배 박스 등

주간: 약 0.3kg
월간: 약 1.2kg
연간: 약 14kg
```

**메시지**
- "이번 달 종이 1.2kg 절약 = A4 용지 240장"
- "올해 종이 14kg 절약 = 나무 0.2그루"

### 5.2 석유 절약

```
플라스틱 1kg 생산 = 석유 약 2L 필요

예시:
월간 플라스틱 2kg 절약
= 석유 4L 절약
= 승용차 연료 50km 주행분
```

**메시지**
- "이번 달 석유 4L를 절약했어요"
- "자동차 연료 1회 주유량의 10%예요"

### 5.3 물 절약

```
플라스틱 1kg 생산 = 물 약 180L 필요

예시:
월간 플라스틱 2kg 절약
= 물 360L 절약
= 욕조 2개 분량
= 2L 페트병 180개
```

**메시지**
- "이번 달 물 360L를 절약했어요"
- "샤워 18번 분량이에요"
- "욕조 2개를 채울 수 있어요"
