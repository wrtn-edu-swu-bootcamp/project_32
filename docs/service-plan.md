# 일회용품 감소 독려 서비스 기획안

## 1. 서비스 개요

### 서비스명
**에코트래커 (EcoTracker)**
부제: 일상 속 작은 변화, 지구를 위한 큰 발걸음

### 서비스 목적
현대 사회에서 무의식적으로 사용되는 일회용품의 사용량을 가시화하고, 사용자가 스스로 일회용품 사용을 줄일 수 있도록 동기를 부여하는 서비스입니다. 단순한 기록을 넘어, 게임화 요소와 환경 영향도 시각화를 통해 지속적인 실천을 이끌어냅니다.

### 비전
- **개인의 작은 실천이 모여 큰 변화를 만든다**는 철학 실현
- 환경 보호를 부담스럽지 않고 재미있는 일상 습관으로 전환
- 데이터 기반의 구체적인 성과를 통해 실질적인 변화 체감

### 배경
- 국내 1인당 플라스틱 소비량 세계 3위 (2021년 기준)
- 일회용품 사용에 대한 경각심은 있으나 실천률이 낮음
- 개인의 작은 실천이 얼마나 큰 영향을 미치는지 체감하기 어려움
- 환경 보호 실천을 지속하기 위한 동기부여 시스템 부재

### 타겟 사용자
**주요 타겟: 전 연령대**

**페르소나 1: 환경에 관심은 있지만 실천이 어려운 직장인**
- 연령: 25-40세
- 특징: 바쁜 일상 속에서 텀블러 사용을 자주 잊어버림
- 니즈: 간편하게 기록하고 동기부여를 받고 싶음

**페르소나 2: 환경 교육에 관심 있는 학부모**
- 연령: 30-50세
- 특징: 자녀와 함께 환경 보호 실천을 하고 싶음
- 니즈: 가족 단위로 함께 참여할 수 있는 활동

**페르소나 3: 환경 운동에 적극적인 MZ세대**
- 연령: 20-30세
- 특징: SNS에 친환경 라이프스타일 공유
- 니즈: 구체적인 데이터와 성과를 확인하고 싶음

**페르소나 4: 작은 실천을 시작하려는 시니어**
- 연령: 50-70세
- 특징: 간단하고 쉬운 방법으로 참여하고 싶음
- 니즈: 복잡하지 않은 UI와 명확한 피드백

## 2. 핵심 가치 제안

### 사용자에게 제공하는 가치

**1. 가시화된 실천**
- 보이지 않던 일회용품 사용량을 구체적인 수치로 확인
- 내 작은 실천이 만드는 환경적 영향을 눈으로 확인

**2. 부담 없는 시작**
- 선택지 기반의 빠른 체크로 1분 안에 완료
- 언제든지 자유롭게 기록 가능한 유연한 시스템

**3. 지속 가능한 동기부여**
- 게임화 요소로 재미있게 참여
- 이중 랭킹 시스템으로 신규/기존 사용자 모두 성취감 제공

**4. 구체적인 피드백**
- "나무 OO그루를 살렸어요" 같은 직관적인 환경 영향도
- 나의 순위와 개선도를 실시간으로 확인

## 3. 주요 기능

### 3.1 일회용품 사용량 체크 시스템

#### 하이브리드 입력 방식
**1단계: 빠른 선택지 체크 (약 30초)**
- 자주 사용하는 일회용품 10-15개를 선택지로 제공
- "오늘 이것을 사용하셨나요?" → 네 / 아니오
- 예시:
  - 플라스틱 컵 커피를 마셨나요?
  - 비닐봉지를 받으셨나요?
  - 배달 음식을 시키셨나요?
  - 페트병 음료를 구매하셨나요?

**2단계: 수동 입력 (선택사항)**
- "오늘 생각나는 다른 일회용품이 있나요?"
- 자유롭게 텍스트로 입력
- 자주 입력하는 항목은 다음에 선택지로 추가 제안

#### 입력 편의성
- 스와이프로 빠르게 네/아니오 선택
- 음성 입력 지원
- 과거 기록 기반 자동 추천

### 3.2 환경 영향도 시각화

#### 즉각적인 피드백
체크 완료 직후 보여지는 화면:

**오늘의 환경 영향**
```
오늘 사용한 일회용품: 5개
- 플라스틱 컵: 2개
- 비닐봉지: 1개
- 배달 용기: 2세트

환경 영향도
🌳 소나무 0.3그루 = 하루 산소 생산량
💨 CO2 약 450g = 스마트폰 225번 충전
🌊 해양 플라스틱 위험도 +5점
```

#### 누적 효과 대시보드
```
이번 달 누적
🌳 나무 9그루를 살렸어요
💨 CO2 13.5kg 절감 = 승용차 70km 주행량
♻️ 재활용 가능 자원 2.3kg 절약

지난달 대비 -23% 개선!
```

#### 비교 시각화
- "같은 사용량이 1년 지속되면?"
- "만약 하루에 1개만 줄인다면?"
- "우리 사용자들이 모두 모이면?" (전체 사용자 누적 효과)

### 3.3 이중 랭킹 시스템

#### 절대량 랭킹: "에코 챔피언"
**누가 가장 적게 사용하는가?**

```
이번 주 에코 챔피언 랭킹
----------------------------
1위 🥇 친환경러123 - 일 평균 1.2개
2위 🥈 그린라이프 - 일 평균 1.8개
3위 🥉 지구지킴이 - 일 평균 2.1개
...
25위 🎯 나 - 일 평균 4.3개

👉 상위 15% 입니다!
전체 평균 5.7개 대비 -24% 적게 사용 중
```

#### 개선도 랭킹: "챌린저 스타"
**누가 가장 많이 줄였는가?**

```
이번 주 챌린저 스타
----------------------------
1위 ⭐ 변화의시작 - 전주 대비 -65%
2위 ⭐ 나 - 전주 대비 -42%
3위 ⭐ 실천왕 - 전주 대비 -38%

🎉 2위 달성! 놀라운 변화에요!
```

#### 랭킹 업데이트 주기
- 주간 랭킹: 매주 월요일 갱신
- 월간 랭킹: 매월 1일 갱신
- 실시간 나의 순위 확인 가능

#### 공정성 확보
- 최소 체크 횟수 충족 시에만 랭킹 참여 (주 3회 이상)
- 이상치 감지 및 필터링
- 신규 사용자 보정 기간 (2주)

### 3.4 게임화 요소

#### 포인트 시스템
**포인트 획득 방법**
- 매일 체크: 10포인트
- 전날 대비 감소: 20포인트
- 연속 체크: +5포인트 (최대 +30)
- 일회용품 제로 데이: 50포인트
- 새로운 뱃지 획득: 100포인트

**포인트 활용**
- 레벨 업 경험치
- 특별 뱃지 언락 조건
- 월간 포인트 랭킹

#### 레벨 시스템
```
Lv.1 새싹 환경러 (0-500p)
Lv.2 초보 실천러 (500-1,500p)
Lv.3 그린 챌린저 (1,500-3,000p)
Lv.4 에코 워리어 (3,000-6,000p)
Lv.5 환경 수호자 (6,000-10,000p)
Lv.6 지구 지킴이 (10,000-20,000p)
Lv.7 에코 마스터 (20,000p+)
```

레벨별 혜택:
- 새로운 통계 페이지 언락
- 특별 칭호 획득
- 커스텀 프로필 아이템

#### 뱃지/업적 시스템 (상세 내용은 gamification.md 참조)
- 첫걸음 뱃지
- 연속 체크 시리즈
- 카테고리별 달성 뱃지
- 특별 이벤트 뱃지

## 4. 차별화 포인트

### 4.1 진입 장벽 최소화
기존 환경 앱들이 복잡한 입력을 요구하는 것과 달리, 선택지 기반의 빠른 체크로 30초 안에 완료 가능

### 4.2 모든 사용자를 위한 랭킹
- 절대량 랭킹: 오래 사용한 사용자의 성과 인정
- 개선도 랭킹: 신규 사용자도 즉시 상위권 가능
- 누구나 성취감을 느낄 수 있는 구조

### 4.3 구체적인 환경 영향 시각화
추상적인 메시지가 아닌 "나무 몇 그루", "CO2 몇 kg" 같은 구체적인 수치로 실감

### 4.4 지속 가능한 동기부여
포인트, 레벨, 뱃지, 랭킹 등 다층적인 보상 시스템으로 장기 사용 유도

## 5. 사용자 경험 플로우 요약

```
1. 앱 실행
   ↓
2. "오늘의 체크 시작하기" 버튼
   ↓
3. 선택지 10-15개 빠르게 체크 (30초)
   ↓
4. "다른 일회용품도 있나요?" (선택)
   ↓
5. 결과 화면
   - 오늘의 환경 영향도
   - 획득한 포인트/뱃지
   - 나의 랭킹 변화
   ↓
6. 대시보드로 이동
   - 주간/월간 통계
   - 누적 효과
   - 전체 랭킹
```

## 6. 핵심 메시지

### 브랜드 톤앤매너
- **긍정적**: 비난이 아닌 격려
- **구체적**: 추상적 메시지가 아닌 수치
- **친근한**: 부담스럽지 않은 일상의 동반자
- **성취적**: 작은 실천의 큰 의미 강조

### 대표 메시지 예시
- "오늘도 지구를 위한 작은 발걸음을 내디뎠어요"
- "당신의 선택이 소나무 한 그루를 살렸어요"
- "어제보다 나은 오늘, 멋진데요!"
- "완벽하지 않아도 괜찮아요. 함께 노력하는 게 중요해요"

## 7. 향후 확장 가능성

### 단기 확장 (3-6개월)
- 친구 초대 및 소셜 공유 기능
- 주간/월간 챌린지 이벤트
- 지역별 랭킹 (우리 동네는?)
- 일회용품 대체제 추천 기능

### 중기 확장 (6-12개월)
- 그룹/팀 챌린지 (회사, 학교, 동호회)
- 리워드 파트너십 (카페 할인 등)
- AI 기반 맞춤형 조언
- 탄소 중립 인증 시스템

### 장기 비전 (1년+)
- 다양한 친환경 실천으로 확장 (에너지, 물 사용 등)
- 기업 CSR 프로그램 연계
- 지자체 환경 정책 데이터 제공
- 글로벌 확장

## 8. 성공 지표 (KPI)

### 사용자 지표
- 월간 활성 사용자 (MAU)
- 주간 체크 빈도
- 사용자 유지율 (1개월, 3개월, 6개월)
- 평균 사용 기간

### 참여 지표
- 일평균 체크 완료율
- 수동 입력 참여율
- 뱃지 획득률
- 랭킹 참여율

### 영향 지표
- 사용자당 평균 일회용품 감소율
- 전체 사용자 누적 절감량
- 개선도 랭킹 참여 비율

## 9. 마무리

에코트래커는 단순한 기록 앱이 아닌, 사용자가 환경 보호를 일상의 즐거운 습관으로 만들 수 있도록 돕는 동반자입니다. 

**"완벽한 환경주의자가 되는 것이 목표가 아니라, 어제보다 조금 더 나은 선택을 하는 것이 목표입니다."**

이 철학을 바탕으로, 누구나 쉽게 시작하고 꾸준히 실천할 수 있는 서비스를 만들어갑니다.
