# 사용자 여정 (User Journey)

## 개요
이 문서는 에코트래커를 사용하는 다양한 시나리오에서의 사용자 경험을 상세히 설명합니다. 사용자가 앱을 처음 접하는 순간부터 장기 사용자가 되기까지의 전체 여정을 다룹니다.

---

## 1. 첫 사용 플로우 (온보딩)

### 1.1 앱 설치 및 시작

**Step 1: 환영 화면**
```
🌍 에코트래커에 오신 것을 환영해요!

일상 속 작은 선택이
지구를 위한 큰 변화가 됩니다

[시작하기]
```

**Step 2: 서비스 소개 (3개 슬라이드)**

슬라이드 1:
```
📊 간편한 체크
선택만 하면 끝!
30초면 오늘의 일회용품 사용을 기록할 수 있어요
```

슬라이드 2:
```
🌳 눈에 보이는 변화
당신의 실천이 나무 몇 그루를 살렸는지
구체적인 숫자로 보여드려요
```

슬라이드 3:
```
🏆 함께하는 즐거움
랭킹, 뱃지, 레벨!
재미있게 환경을 지켜요
```

**Step 3: 기본 정보 입력 (선택사항)**
```
더 나은 경험을 위해 알려주세요 (건너뛰기 가능)

닉네임: [____________]

주로 활동하는 지역:
○ 서울
○ 경기
○ 기타 지역

[다음으로] [건너뛰기]
```

**Step 4: 관심 카테고리 선택**
```
어떤 일회용품을 주로 사용하시나요?
(나중에 변경 가능)

☑️ 음식 관련 (배달, 카페 등)
☑️ 음료 관련 (페트병, 캔 등)
☑️ 포장재 (비닐봉지, 택배 등)
☐ 위생용품 (마스크, 물티슈 등)
☐ 기타

[선택 완료]
```

**Step 5: 첫 체크 유도**
```
🎉 준비 완료!

그럼 오늘 하루를 돌아볼까요?
첫 체크를 시작하면 10포인트를 드려요!

[오늘의 체크 시작하기]
```

## 6. 동기부여 포인트 및 리텐션 전략

### 6.1 각 단계별 동기부여

**첫 체크 후 (1일차)**
- 즉각적인 보상 (포인트, 뱃지)
- 구체적인 환경 영향도 숫자
- 다음 체크 유도 ("내일도 확인해볼까요?")

**초기 사용 (1주일)**
- 연속 체크 보너스
- 빠른 레벨업 경험
- 주간 리포트로 성과 확인

**정착 단계 (1개월)**
- 랭킹 시스템 참여
- 월간 달성 뱃지
- 개인 맞춤 인사이트

**장기 사용자 (3개월+)**
- 누적 환경 기여도 강조
- 고난이도 뱃지 챌린지
- 커뮤니티 기여 (데이터 제공)

### 6.2 푸시 알림 전략

**매일 리마인더 (사용자 설정 시간)**
```
⏰ 오늘의 체크 시간이에요!
🔥 3일 연속 체크 중 - 끊기지 않게 해요!
```

**동기부여 메시지**
```
💪 어제보다 2개 줄였어요!
   이번 주도 화이팅!
```

**랭킹 변화 알림**
```
🏆 3계단 상승! 현재 25위
   상위 15%에 진입했어요!
```

**뱃지 달성 임박**
```
🏅 30일 연속까지 3일 남았어요!
   조금만 더 힘내요!
```

**특별 이벤트**
```
🎄 12월 특별 챌린저!
   이번 달 평균 3개 이하 달성하면
   특별 뱃지를 드려요!
```

### 6.3 이탈 방지

**3일 미접속 시**
```
보고 싶었어요! 👋
그동안 어떠셨나요?

지난주 당신의 기록:
🌳 나무 2.1그루 보호
💪 랭킹 25위 유지 중

작은 습관이 큰 변화를 만들어요
```

**1주일 미접속 시**
```
💚 당신의 빈자리가 느껴져요

지금까지의 성과:
✓ 총 체크: 23일
✓ 나무 15그루 보호
✓ 최고 랭킹: 18위

다시 시작하는 건 언제나 환영이에요!
[오늘 체크하기]
```

## 8. 사용자 여정 요약

### 신규 사용자 (1일차)
온보딩 → 첫 체크 → 즉각 피드백 → 재방문 유도

### 초기 사용자 (1주)
일일 체크 습관화 → 연속 보너스 → 주간 리포트 → 랭킹 참여

### 활성 사용자 (1개월)
안정적 체크 → 뱃지 수집 → 랭킹 경쟁 → 개인 목표 설정

### 충성 사용자 (3개월+)
장기 트렌드 확인 → 누적 기여도 → 고난이도 챌린지 → 커뮤니티 기여

## 9. 핵심 경험 원칙

1. **즉각적 피드백**: 모든 행동에 즉시 반응
2. **구체적 성과**: 추상적이지 않은 명확한 숫자
3. **긍정적 강화**: 비난이 아닌 격려와 축하
4. **점진적 개선**: 완벽보다는 어제보다 나은 오늘
5. **재미있는 실천**: 게임화로 지속 가능한 동기부여
